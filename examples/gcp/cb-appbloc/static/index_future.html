Building blocks (“blocs”)

Composable modules you can adopt independently or as a full platform.

AppBloc

The simplest way to deploy apps to Kubernetes — opinionated namespaces, deployments, services, ingress, and managed TLS. Scales from hello-world to production with GitOps-friendly defaults.

ObsBloc

Prometheus + Grafana + Loki/Tempo with dashboards, alerts, and cost-aware retention.

SearchBloc

Open-source search at scale with sensible sharding, snapshots, and SLO-backed tuning.

Deploy in minutes

Use Terraform to install a bloc into an existing cluster. Everything ships with examples and dashboards.

# Example: install ObsBloc
module "obsbloc" {
  source  = "github.com/cloudbloc-io/obsbloc//terraform"
  project = "acme-prod"
  cluster = "gke-us-central1"
  region  = "us-central1"
  # opinionated defaults; override as needed
}
